# 🚀 Daily DSA + LLD Practice App (Node.js)

## 📌 Overview

This project is a simple **Node.js-based local development environment** designed to:

* Practice **DSA (Data Structures & Algorithms)** daily
* Work on **LLD (Low-Level Design) projects**
* Run everything locally on `http://localhost:3000`
* Dynamically switch between different project files using a single `index.html`

---

## ⚙️ Features

* 🔁 Serve HTML file via Node.js server
* 🧠 Practice DSA problems with separate JS files
* 🏗️ Build and test LLD mini-projects
* 🔄 Dynamically switch running scripts using `<script>` tag in `index.html`
* ⚡ No need to restart server for every change
